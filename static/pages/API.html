Hate Speech Detection - API

Instructions on how to use the Hate Speech Detection API:

  1. Make a POST request to the API endpoint with the following JSON payload: 
    {
    "text": "Your text to be checked for hate speech"
}
  2. The API will return a JSON response with the prediction result. Example response: 
    {
    "text": "Your text to be checked for hate speech",
    "prediction": "Hate Speech" // or "Non-Hate Speech"
}
  • Install the necessary dependencies: pip install requests
  • Use the following code snippet to interact with the Hate Speech Detection API: 
    import requests

API_ENDPOINT = "http://endpoint/predict"

text_to_check = "Your text to be checked for hate speech"
payload = {"text": text_to_check}

response = requests.post(API_ENDPOINT, json=payload)
prediction = response.json()["prediction"]
print("Prediction:", prediction)
